Para ejecutar el programa debe tener en cuenta
los siguientes puntos

Requisitos 

    Tener instalado python3
    tener instalado el modulo persistent (haber instalado previamente pip3)

Instalacion

    Python3: sudo apt-get install python3
    pip3: sudo apt-get install python3-pip3
    modulo persistent: pip3 install persistent

Ejecucion del programa

    Para iniciar la aplicacion debe ejecutarse el script RcsApp.py desde la terminal
    =>$ python3 RcsApp.py

Ejecucion del programa TK

    Para iniciar la aplicacion debe ejecutarse el script RcsAppVista.py desde la terminal
    =>$ python3 RcsAppVista.py
